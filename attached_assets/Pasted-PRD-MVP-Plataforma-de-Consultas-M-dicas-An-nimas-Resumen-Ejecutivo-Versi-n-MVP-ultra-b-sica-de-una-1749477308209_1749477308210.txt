PRD MVP - Plataforma de Consultas Médicas Anónimas
Resumen Ejecutivo
Versión MVP ultra-básica de una plataforma web para consultas médicas anónimas entre profesionales de la salud. Enfoque en funcionalidad esencial y estabilidad.
Features del MVP (3 características principales)
1. Autenticación Básica

Login simple: Email + contraseña hardcodeada
Credenciales de prueba:

Email: doctor@hospital.com
Password: 1234


Sin registro: Usuario pre-existente en el sistema
Sesión básica: Mantener usuario logueado durante la sesión

2. Dashboard de Casos

Lista de casos: Vista simple con casos existentes
Estados básicos: Solo 3 estados

Nuevo (azul)
En revisión (amarillo)
Resuelto (verde)


Información mínima por caso:

Título del caso
Estado actual
Fecha de creación
Botón "Ver detalles"



3. Crear Nuevo Caso

Formulario simple con campos esenciales:

Título del caso (obligatorio)
Sexo del paciente (M/F/Otro)
Rango de edad (dropdown: 0-18, 19-35, 36-50, 51-65, 65+)
Descripción del caso (textarea)
Consulta médica (textarea)
Nivel de urgencia (Baja/Media/Alta)


Validación básica: Campos obligatorios
Confirmación: Mensaje de éxito + redirect al dashboard

Pantallas del MVP
1. Login (/login)

Formulario con email y password
Botón "Iniciar sesión"
Validación básica
Redirect a dashboard si es exitoso

2. Dashboard (/dashboard)

Header con "Consultas Médicas" + botón "Cerrar sesión"
Botón destacado "Nuevo Caso"
Lista de casos existentes
Cada caso muestra: título, estado (con color), fecha

3. Nuevo Caso (/nuevo-caso)

Formulario con todos los campos del modelo
Botones "Guardar" y "Cancelar"
Validación en frontend
Confirmación de éxito

4. Detalle del Caso (/caso/:id)

Vista de solo lectura con toda la información
Botón "Volver al dashboard"
(Futuro: aquí iría el chat)
